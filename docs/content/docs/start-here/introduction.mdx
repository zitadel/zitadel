# Introduction

ZITADEL is a cloud-native identity management system that provides secure authentication and authorization for your applications.

## Value Proposition

- **Multi-tenancy**: Built from the ground up for B2B SaaS applications.
- **Open Source**: Run it anywhere, inspect the code, and contribute.
- **Audit Trail**: Complete history of all changes thanks to Event Sourcing.
