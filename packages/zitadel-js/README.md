# @zitadel/zitadel-js

ZITADEL JavaScript SDK — isomorphic core. Zero framework dependencies. Runs in Node.js ≥18, Edge runtimes, and browsers.

## Installation

```sh
npm install @zitadel/zitadel-js
```
